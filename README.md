# snake-nx
The classic Snake game, as a homebrew app for Nintendo Switch.

## Installation
Requires a switch running CFW.
- Place `snake.nro` in `/switch` on your Switch's SD card.
- Launch the app through HBMenu.

## Controls
- **DPAD** - Steer snake
- **MINUS** - Pause the game
- **PLUS** - Exit

## Building
- See [here](https://switchbrew.org/wiki/Setting_up_Development_Environment) for instructions on preparing the development environment. 
- Run `make` in the root of this repo.
